wandb使用

1.需要到wandb注册个账号 ,然后登入。

2.进入setting ,下滑

3.点击new key, 然后复制这个key

4.登入服务器 , 进入python 虚拟环境, 输入 pip install wandb

5.安装完毕后,输出 wandb init , 然后黏贴key

6.在python文件头 import wandb

7.在主函数 wandb.init(project=”project name “)

  • project name 随便起

8.在需要上传数据的地方 wandb.log(map_var)

  • map_var 这个是字典类型的数据

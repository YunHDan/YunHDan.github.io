# Javascript зҹҘиҜҶж ‘

# еҹәзЎҖ

# дҪңз”ЁеҹҹпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡеұҖйғЁдҪңз”ЁеҹҹгҖҒе…ЁеұҖеҸҳйҮҸгҖҒеҸҳйҮҸз”ҹе‘Ҫе‘Ёжңҹ

йңҖиҰҒзү№еҲ«е…іжіЁпјҡ

  • еҰӮжһңеҸҳйҮҸеңЁеҮҪж•°еҶ…йғЁжІЎжңүеЈ°жҳҺпјҲжІЎжңүдҪҝз”Ё var е…ій”®еӯ—пјүпјҢеҲҷиҜҘеҸҳйҮҸдёәе…ЁеұҖеҸҳйҮҸгҖӮ
  • жүҖжңүе…ЁеұҖеҸҳйҮҸеұһдәҺ Window еҜ№иұЎгҖӮ

# еҮҪж•°пјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡеҮҪж•°зҡ„иҜӯжі•гҖҒеҸӮж•°дј йҖ’гҖҒиҝ”еӣһеҖјгҖҒ

йңҖиҰҒзү№еҲ«е…іжіЁпјҡ

  • return еҸҜжҺҘиҝ”еӣһеҖјпјҢиҝ”еӣһеҖјеҸҜжңүеҸҜж— гҖӮ
  • еңЁеҮҪж•°еҶ…йғЁеЈ°жҳҺзҡ„еҸҳйҮҸпјҲдҪҝз”Ё var пјүжҳҜеұҖйғЁеҸҳйҮҸгҖӮеҮҪж•°иҝҗиЎҢе®ҢжҜ•пјҢеұҖйғЁеҸҳйҮҸе°ұдјҡиў«еҲ йҷӨгҖӮ

# ж•°жҚ®зұ»еһӢпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡж•°жҚ®зұ»еһӢзҡ„еҲҶзұ»гҖҒеҠЁжҖҒзұ»еһӢзҡ„жҰӮеҝөгҖҒеӯ—з¬ҰдёІзұ»еһӢгҖҒж•°еӯ—зұ»еһӢгҖҒеёғе°”зұ»еһӢгҖҒж•°з»„зұ»еһӢгҖҒеҜ№иұЎзұ»еһӢгҖҒ undefined дёҺ null гҖҒ new е…ій”®еӯ—еЈ°жҳҺеҸҳйҮҸзұ»еһӢгҖӮ

йңҖиҰҒзү№еҲ«е…іжіЁпјҡ

  • еҸҜд»Ҙз”Ё typeof жҹҘзңӢеҸҳйҮҸзҡ„ж•°жҚ®зұ»еһӢгҖӮ
  • Symbol ж•°жҚ®зұ»еһӢиЎЁзӨәзӢ¬дёҖж— дәҢзҡ„еҖјгҖӮ
  • еҜ№иұЎзҡ„еЈ°жҳҺеҸҜд»Ҙи·Ёи¶ҠеӨҡиЎҢпјҢз©әж је’ҢжҠҳиЎҢж— е…ізҙ§иҰҒгҖӮ
  • еҜ№иұЎзҡ„еұһжҖ§жңүдёӨз§ҚеҜ»еқҖж–№ејҸпјҲиҺ·еҸ–ж–№ејҸпјүгҖӮ

# еҸҳйҮҸпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡеЈ°жҳҺеҸҳйҮҸгҖҒдёҖжқЎиҜӯеҸҘеЈ°жҳҺеӨҡдёӘеҸҳйҮҸгҖҒ let дёҺ c onst гҖӮ

йңҖиҰҒзү№еҲ«жіЁж„Ҹпјҡ

  • Javascript зҡ„иҜӯеҸҘе’ҢеҸҳйҮҸйғҪеҜ№еӨ§е°ҸеҶҷж•Ҹж„ҹгҖӮ
  • еҸҳйҮҸе°ҪйҮҸд»Ҙеӯ—жҜҚејҖеӨҙгҖӮ
  • еҰӮжһңйҮҚж–°еЈ°жҳҺ Javascript еҸҳйҮҸпјҢиҜҘеҸҳйҮҸзҡ„еҖјдёҚдјҡдёўеӨұгҖӮ
  • еЈ°жҳҺеҸҳйҮҸж—¶жңӘжҢҮе®ҡеҖјпјҢеҲҷй»ҳи®Өдёә undefined гҖӮ

# жіЁйҮҠпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡеҚ•иЎҢжіЁйҮҠж–№жі•гҖҒеӨҡиЎҢжіЁйҮҠж–№жі•гҖҒиЎҢжң«жіЁйҮҠж–№жі•гҖӮ

# иҜӯеҸҘпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡеҲҶеҸ·дҪңз”ЁгҖҒе…ій”®еӯ—иЎЁгҖҒеҜ№д»Јз ҒиЎҢиҝӣиЎҢжӢҶиЎҢгҖӮ

йңҖиҰҒзү№еҲ«жіЁж„Ҹзҡ„пјҡ

  • з”ЁеҲҶеҸ·жқҘз»“жқҹиҜӯеҸҘжҳҜеҸҜжңүеҸҜж— зҡ„гҖӮ
  • з©әж јз”ЁдәҺеўһеҠ еҸҜиҜ»жҖ§пјҢ Javascript дјҡиҮӘеҠЁзңҒз•ҘеӨҡдҪҷз©әж јгҖӮ

# иҫ“еҮәпјҡиҸңйёҹж•ҷзЁӢ

ж¶өзӣ–зҹҘиҜҶпјҡ window.alert() гҖҒ getElementById() гҖҒ innerHTML гҖҒ write() гҖҒ console.log() гҖӮ

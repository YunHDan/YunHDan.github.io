# 已入库的番剧

已经看完了的番剧。

# 古早 —— 初中时期

  • 海贼王：这个到现在也在关注。
  • 罪恶王冠：看完了结局难受了几天。
  • 刀剑神域第一季：当时感觉设定新奇。

# 高中时期

  • fate zero & fate stay night ：记忆混乱。
  • 异度侵入：史诗级烧脑推理。
  • Re0 第一季、第二季：从窝囊男主到顶天立地男子汉。

# 大一

  • 孤独摇滚：乐队的日常欢欢乐乐就由专业的芳文社打造！
  • 电锯人：电次君别再被蒙在鼓里啦！
  • 鬼灭之刃第一季：变强吧炭治郎！
  • 鬼灭之刃无限列车：大哥没有输！
  • 夏日重现：体验和高智商反派战斗是什么感觉。

# 大二

  • 进击的巨人全四季：“动漫中的文学”？
  • 鬼灭之刃游郭篇：极致到颅内高潮的超高经费打戏。
  • 鬼灭之刃刀匠村篇：妹妹能见太阳啦！
  • 赛博朋克 - 边缘行者：浪漫或许是送对方一个月球旅行。
  • 你想活出怎样的人生：宫崎骏收官作。
  • 小林家的龙女仆：想要成为小林被龙龙们包围。
  • 不死少女杀人笑剧：东方侦探来到福尔摩斯的世界。
  • 石纪元第一季、第二季：用科学重建人类世界。

# 大二暑假

  • 哭泣少女乐队：乐队番中的《解忧杂货店》。

  • 咒术回战第一季、怀玉、涩谷：一个看了能让你变帅的番。

  • 青春猪头少年不会梦到兔女郎学姐：人活着就是为了樱岛麻衣！

  • 没落要塞：即使见不到什么希望，但我只是以最大的努力去抗争！

  • 葬送的芙莉莲：呀，今天芙莉莲大人这么早起！

# 大三上
